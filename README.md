## About the Project

This project is a modern **Real Estate Website UI** designed to showcase property listings and provide users with an easy way to contact agents.

The website includes smooth UI animations using **Framer Motion** and a fully responsive layout built with **Tailwind CSS**.

The contact form is integrated with **Web3Forms** for easy form submission without needing a custom backend.

---

## Project Overview

The goal of this project is to create a modern and responsive **real estate landing page** that provides a smooth user experience.

### Objectives

* Create a clean and modern **property listing interface**
* Implement **Framer Motion animations**
* Build a **responsive layout**
* Enable **contact form submission with Web3Forms**
* Provide fast loading performance
